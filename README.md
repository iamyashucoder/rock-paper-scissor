# Rock Paper Scissors Game

A simple Python terminal game where the user plays Rock, Paper, Scissors against the computer. The game uses ASCII art to display both the user's choice and the computer's random choice.

## Features

- Play Rock, Paper, Scissors in the terminal
- Random computer choice
- ASCII art for Rock, Paper, and Scissors
- Win, lose, and draw result messages
- Beginner-friendly Python code

## How to Run

Make sure Python is installed on your computer.

Run the game with:

```bash
python main.py
